#### DF-004

### Sumary

Benefits Dashboard - POST/PUT - Salary

### Type

BE

### Description

According to the requirements, all employees should have a fixed salary of 52000.
However:

- POST correctly sets salary to 52000, ignoring any value sent by the client.

- PUT allows updating the salary to any value, overriding the fixed amount.

This contradicts the stated rule and results in inconsistent data between POST and PUT operations.

POST/PUT API: https://wmxrwq14uc.execute-api.us-east-1.amazonaws.com/Prod/api/Employees

###### Steps to reproduce:

1. Call POST API
2. Salary is 52000
3. Call PUT API with id from POST body and different salary amount
4. Salary amount in PUT respond and FE Benefits Dasboard is not 52000

### Severity

Medium
