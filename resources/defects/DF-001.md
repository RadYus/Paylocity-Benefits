#### DF-001

### Sumary

Benefits Dashboard - Switched first/last name

### Type

FE

### Description

The Benefits Dashboard displays the employee’s first and last name incorrectly.

The API returns the values in the correct fields, but the frontend maps them incorrectly and swaps them.

###### Steps to reproduce:

1. Log in to the frontend application
2. On the "Benefits Dashboard" page, click the "Add Employee" button
3. Enter any Employee First Name, Last Name & Dependent information
4. Click the "Submit" button
5. First and Last name are switched in the UI

### Severity

High
